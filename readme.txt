== Installation ==

Installing this plugin as regular wordpress plugin. First you need to install Visual Composer plugin then install Team Member Group For Visual Composer.

<h2>This addon plugin works like other page builder addon plugin.</h2>
<p>Go to new page then click on Visual Composer builder.</p> 
<p>then click on Add Element you should see Visual Composer Extensions menu as Name "Team Member" on Manu Top.</p>
<p>click on it you will see Team member group Items Menu, Just Click On it. </p>
<p>All Setting Panel FOr Team member Group Will Be open. </p>
<p>Open it and browse an Member image and customize according to your needs.</p>
